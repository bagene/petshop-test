# BACS Implementation

## Installation

1. Make sure to have the package on your composer repositories
```
"repositories": [
    {
        "type": "path",
        "url": "packages/bacs"
    }
]
```
2. Run `composer require bagene/bacs:dev-main`
3. Routes will be automatically configured on your app

## Routes

* /api/v1/bacs
